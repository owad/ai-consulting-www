# AI Konsultacje - Strona wizytówkowa

Nowoczesna, minimalistyczna strona wizytówkowa dla firmy świadczącej usługi konsultacji AI.

## Charakterystyka

- **Design:** Nowoczesny, czytelny, gradient indigo/cyan
- **Responsywny:** Działa na wszystkich urządzeniach
- **Single-page:** Wszystko na jednej stronie (hero, usługi, o nas, CTA)
- **Bez zależności:** Czysty HTML/CSS, brak frameworków
- **Szybki:** Minimalistyczny kod, szybkie ładowanie

## Sekcje

1. **Header (Hero)** - Gradient background z efektem wave
2. **Usługi** - Grid 6 kart z ikonami emoji
3. **O nas** - Krótki opis firmy
4. **CTA** - Wezwanie do działania z przyciskiem kontaktu
5. **Footer** - Stopka

## Kolory

- Primary: `#6366f1` (indigo)
- Secondary: `#0ea5e9` (cyan)
- Dark: `#0f172a` (slate)
- Gray: `#64748b`
- Light: `#f1f5f9`
